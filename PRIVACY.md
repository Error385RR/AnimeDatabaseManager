# Privacy Policy

## Overview

Anime Database Manager is a local application designed to manage a personal anime database.

## Data Collection

This application does not collect, upload, or transmit personal information to any external server operated by the developer.

## Local Storage

All user data, database files, and configuration files are stored locally on the user's device.

## MyAnimeList API

The application may communicate directly with MyAnimeList services through their API. API credentials and authentication tokens are stored locally and are only used for accessing MyAnimeList services.

## Third Parties

This application does not provide user data to third parties.

## Contact

For questions regarding this application, please use the repository issue tracker.
